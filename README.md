## Setup

1. Sign up at [authors-voice.com](https://authors-voice.com)
2. Generate an API key at [Voice Studio > API Keys](https://authors-voice.com/voice?tab=api-keys)
3. Set your key: `export AV_API_KEY="av_live_YOUR_KEY_HERE"`
4. Say "setup voice" to get started

## What it does

- **Import** writing from Google Drive, Notion, URLs, or raw markdown
- **Analyze** your voice across 6 linguistic categories (diction, syntax, punctuation, rhetoric, discourse, idiolect)
- **Apply** your voice to rewrite AI-generated content or generate new content that sounds like you
- **Detect** and eliminate AI patterns that trigger detection tools

18 tools. One skill. Your voice.
